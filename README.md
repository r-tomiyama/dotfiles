# peko の環境構築

## コマンド

### 全てを適用する

```sh
sh apply.sh
```

### homebrewでライブラリをインストール

```sh
make install_library_with_homebrew
```

### VScode プラグインをインストール

```sh
make install_vscode_plugin
```

## 初期構築

1. homebrew をインストール
1. `brew install git`
1. git コマンドで SSH接続できるようにする
1. このリポジトリをclone
1. homebrewでライブラリをインストール
